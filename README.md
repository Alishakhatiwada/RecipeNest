## Setup Instructions
npm install
npm run dev

## Technologies Used
Node.js
Express
MongoDB
JWT
Multer
bcrypt

## Features Implemented
- User Registration & Login
- JWT Authentication
- Role-based Authorization
- File Upload (Profile + Recipe)
- Protected Routes
- Recipe CRUD
